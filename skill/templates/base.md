# Profile Base Template

## Questions to Ask

Before filling this template, gather:

1. **Contact:** Email, LinkedIn URL, location
2. **Current role:** Title, company, duration, key responsibilities
3. **Previous roles:** 2-3 most relevant (title, company, duration, achievements)
4. **Top skills:** 5-7 core competencies, split into primary vs secondary
5. **Languages:** Native, professional, working proficiency
6. **Education:** Degrees, institutions, years
